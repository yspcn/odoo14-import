  • Easy to import product variant eCommerce category from CSV/Excel.
  • You can add extra product media in the eCommerce category.
  • You can import product variant (image, price, quantity, stock, etc) from CSV or Excel by barcode, internal reference, or name.
  • You can create or update product variants from CSV/Excel.
  • You can identify the same name product/product variant by "Unique Identification" fields.
  • You can import product images by image URL or local image path.
  • Easy to import products with stock on-hand quantity.

About Fields

Column Product Model Field Name Description Required
A Product template Unique Identification You can identify the same name product/product variant by "Unique Identification" field. Yes
B Product template Name Define the product/product variant name. Yes
C Product template Can Be Sold? It takes true by default if not provided. No
D Product template Can be Purchased? It takes true by default if not provided. No
E Product template Product Type In the "Product type" take consumable by default if not provided. No
F Product template Category It takes default category if not provided. No
G Product template Ecommerce Categories You have to choose eCommerce product category. No
H Product template Unit of Measure Take the same name as ERP in the "UOM". No
I Product template Purchase Unit of Measure Take the same name as ERP in the "UOM". No
J Product template Customer Taxes Taxes are separated by comma(,). You have to provide customer tax here. No
K Product template Vendor Taxes Taxes are separated by comma(,). You have to provide vendor tax here. No
L Product template Description for Customers You have to write a product description here. No
M Product template Invoicing Policy Enter the same text of selection field the same as ERP in the"Invoice policy" otherwise by default it takes ordered quantities. No
N Product template Sales Price You have to enter the product sales price here. No
O Product template Cost You have to enter the product cost here. No
P Product template Extra Product Media You can provide extra media for the product. No
Q Product template Is Published It takes default category if not provided. No
R Product variant variant Attributes You have to enter the product variant attributes. No
S Product variant Attribute Values You have to enter the product attribute values. No
T Product variant Internal Reference You have to enter the product internal reference number. No
U Product variant Barcode You have to enter the product barcode number. No
V Product variant Weight You have to enter the product weight. No
W Product variant Volume You have to enter the product volume. No
X Product variant Qty On Hand You have to enter the product on hand quantity. No
Y Product variant Image path/url Enter the complete URL or local path in the "Image path/URL". No
Z Product variant Extra Variant Media Enter the extra product variant media here. No
